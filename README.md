# HExHTTP
Header Exploitation HTTP

### Beta version


## Usage

	usage: hexhttp.py [-h] [-u URL] [--full]

  	-h, --help  show this help message and exit
  	-u URL      URL login to test [required]
  	--full      To display full header


## Features

- Check CPDoS
- Localhost header
- Ngninx - Envoy - Apache
- CND Analysis
